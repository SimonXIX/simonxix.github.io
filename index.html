open-source systems at SOAS Library

As part of a project team at SOAS, University of London, I implemented a unique network of open-source library applications comprising library management system (OLE), catalogue and discovery interface (VuFind), digital library management software (SobekCM), and integrations between these systems. All our code was made available on GitHub and an article about the implementation was published in Ariadne, issue 74, in June 2015.

Open Access Button interlibrary loan project

While working at Imperial College London, I worked on a Jisc R&D project assessing the feasibility of using Open Access Button functionality to aid in discovery and promotion of open-access content by integrating the Open Access Button into existing interlibrary loan systems in libraries. My team was responsible for investigating integrating the Open Access Button with Ex Libris Alma via Alma's RESTful API. The full findings of the project are available at this link with the Imperial-specific report at this link.

Qatar Digital Library

As part of a project team at The British Library and in partnership with the Qatar Foundation, I helped to design and build the Qatar Digital Library. I worked as Business Analyst assessing technical requirements for the business and refining the workflow for digitisation and cataloguing.

search and discovery

I specialise in search and discovery systems for libraries (with a particular focus on open-source systems) and UX research into users' experiences of search and discovery in the UK. I have implemented several VuFind discovery interfaces and have worked extensively using the open-source search engines Apache Solr and Elasticsearch. I have also maintained and configured Ex Libris Primo and III Encore search systems.

tech for Radical Librarians Collective

As a volunteer for the Radical Librarians Collective - an autonomous workers collective focused on radical politics in librarianship - I have implemented technical solutions for communication, collaborative working, and information security. Collaboratively the Tech Committee of RLC has implemented and maintains a Sandstorm server for collaborative working and document mangement, an Open Journal Systems journal management site for the Journal of Radical Librarianship, and a Tor bridge relay for connecting to the Tor network.
